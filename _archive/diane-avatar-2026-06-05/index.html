Diane Avatar Project

Experiments in giving an AI agent a visual presence.

This is a collection of prototype iterations exploring different approaches to rendering an avatar for OpenClaw-based AI agents. Each version tries a different rendering technique or interaction model. The goal is to find what works before committing to a production approach.

Source: github.com/dianeclawd/diane-avatar

Prototypes

Preview Details
v3 — Sprite Dashboard
active · February 26, 2026
AI-generated pixel art character on the same real-time monitoring dashboard. 11 pose frames mapped to agent states. No Live2D — just sprite swaps with CSS transitions. Lighter runtime, different aesthetic.
v2 — Live Dashboard
active · February 25, 2026
Live2D character (Haru) on a real-time monitoring dashboard. System vitals, agent state machine, activity feed. Data relayed through Azure Web PubSub. Requires GitHub sign-in for cloud access.
v1 — Diorama
paused · February 22, 2026
Live2D character in a Three.js layered scene. First experiment with compositing a 2D avatar into a styled 3D environment. Cursor tracking, expression and motion controls.
